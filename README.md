# Superwish

A Wishlist web app using
- React
- Redux (A bleeding-edge Flux Implementation)
- Webpack
- Universal app (Isomorphic / Server Rendering)
- ES6/7 (through Babel)
- MongoDB
- Express.js

App live at http://superwish.herokuapp.com/
